# Cymunio Scanning Tool

A cybersecurity learning project inspired by OWASP Nettacker, focused on understanding network scanning, reconnaissance, and security analysis workflows.

---

## 📌 About This Project
This project is a customized learning version of an open-source network security scanner. It is used strictly for educational and research purposes.

---

- Understand how network scanning tools work
- Learn reconnaissance techniques
- Explore vulnerability detection concepts
- Practice cybersecurity tool structuring and deployment
